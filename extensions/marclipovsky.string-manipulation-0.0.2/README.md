# String Manipulation for VSCode

This extension provides string manipulation commands for any selected text as well
as multiple selections.

Current string functions available:

1. camelize
1. capitalize
1. classify
1. dasherize
1. decapitalize
1. humanize
1. join lines
1. reverse
1. slugify
1. swap case
1. titlize
1. underscored
